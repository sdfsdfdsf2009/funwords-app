# FunWords - 词汇学习游戏

一个现代化的词汇学习游戏，使用 8-bit 像素风格，帮助用户通过有趣的方式学习新词汇。

## 功能特点

- 🎮 三种题型：选择题、拼写题、听力题
- 🎵 Web Speech API 发音功能
- 📱 响应式设计，支持移动端
- 💾 本地存储学习进度
- 🎨 现代化蓝绿渐变界面

## 技术栈

- HTML5 语义化标签
- CSS3 现代化设计
- JavaScript ES6+
- Web Speech API
- LocalStorage
